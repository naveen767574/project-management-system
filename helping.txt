⭐ Your Daily Git Workflow (Easy Version)

Whenever you make changes in VS Code, do these three commands:

1️⃣ See changed files
git status
2️⃣ Add all updated files
git add .
3️⃣ Commit with a message
git commit -m "your message here"
4️⃣ Push to GitHub
git push

That's it.
Just 4 simple commands, always the same.


⭐ Bonus: What Your Teammates Do

When your teammates clone your repo:

git clone <your-repo-link>

When they want the latest updates:

git pull

When they make changes:

git add .
git commit -m "updated resume parser"
git push

That’s all.
